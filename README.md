Week 2 – pfSense Firewall Deployment & Security Configuration
Overview

This project was completed as part of my Cybersecurity Blue Team / SOC Analyst Internship at ITSimplera Institute.

The objective of this project was to deploy and configure a secure firewall environment using pfSense, implement intrusion detection using Suricata IDS/IPS, configure pfBlockerNG for GeoIP filtering and DNSBL website blocking, and validate firewall security controls.

Objectives
Deploy pfSense Firewall
Configure WAN and LAN Interfaces
Install Suricata IDS/IPS
Install pfBlockerNG
Configure GeoIP Country Blocking
Configure DNSBL Website Blocking
Configure Administrator WebGUI Access
Verify Firewall Logs and Alerts
